# Operator Boundary

Public operator documentation describes boundaries, not private runbooks.

Allowed public content:

- supported chain IDs and endpoints
- relayer trust boundary
- public selector and pool binding
- known limitations
- incident reporting channel

Private content that must stay out of the public repo:

- funding records
- key rotation plans
- provider namespace IDs
- deployer private configuration
- smoke records
- mainnet broadcast packets
- private authorization records
