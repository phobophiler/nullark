# Nullark Docs Source

This root docs tree contains public documentation for open review.

- `public/` explains the public product surface.
- `developers/` explains runtime config, contracts, proofs, and artifact verification.
- `security/` explains the threat model, known limitations, and reporting process.
- `operators/` explains public operational boundaries without exposing private runbooks.

Private deployment approvals, funding records, key rotation material, live mainnet smoke evidence, and Cloudflare state do not belong in the public docs tree.
