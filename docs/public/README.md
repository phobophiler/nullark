# What Nullark Is

Nullark is a fixed-denomination ETH note system on MegaETH. A user deposits into `NullarkPool`; the app generates and handles note material through browser state and wallet-gated recovery; the user later exits to a public recipient with a proof-checked withdrawal.

Nullark does not make a production anonymity claim. Deposits, withdrawals, recipients, fixed amounts, timing, commitments, nullifiers, encrypted-note bytes, and the submitting wallet or relayer remain public.
