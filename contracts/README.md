# Contracts

Nullark contracts are Solidity sources for the fixed-denomination native-ETH pool and verifier adapter.

- `src/` contains production contract source.
- `test/` contains Foundry tests and synthetic fixtures.
- `script/` contains deployment and verification scripts. Public scripts must be dry-run or template-oriented by default.

Mainnet addresses and verifier bindings are published in `../public-artifacts/current.json`.
