# Approved Nullark v1.1 Proving Bundle

This directory is the immutable repo-local approved-artifact boundary for the historical Nullark v1.1 MegaETH mainnet proving records.

Readiness validators must hash files from this bundle, not mutable regeneration output under `circuits/build/`. The artifact hashes intentionally match the approved v1.1 provenance record; this directory changes the path boundary, not the approved bytes, owner approval, trusted setup meaning, or runtime status.

The bundle includes the approved Powers of Tau bytes, circuit symbols, R1CS, WASM, final zkeys, verification keys, and generated Solidity verifier sources needed for v1.1 readiness validation.
